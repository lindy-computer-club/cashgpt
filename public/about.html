Let's Talk Cash is our club's take on Girls Who Code's AI4FinancialGood challenge.

The project includes pages supporting the main feature of CashGPT.

CashGPT integrates OpenAI's API with ChatGPT version 3.5, and is customized to give topical and relevant advice when given inputs for the user's financial details.

The code used for the website is open-source and can be found on GitHub.


To see CashGPT in action without a OpenAI key, check out our examples page.
